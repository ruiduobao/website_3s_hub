# 分栏介绍

::: tip 建站原因
目前没有以遥感资源**内容为导向**的网站，找数据对于遥感初学者是个比较费劲的事。
:::

::: tip 本站内容

1.只提供**官方下载**方法。不提供直接的数据，授人以鱼，不如授之以渔。

2.不做爬虫类的数据教程，不做涉密数据的教程，不做具有知识产权数据的免费获取教程，
:::
